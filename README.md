# 1ST Challenge - Time Series Classification
- Developing a recurrent neural network able to perform time series classfication given a collection of multivariate time series.

In this assignment, a collection of multivariate time series was given. Each sequence consisted of 180 time steps and contained measurements from several input channels. The objective of this task is to perform time series classification — that is, to assign each entire sequence to one of three possible classes.

⚠️ Please note that, as specified in the Rules section, the use of pretrained models is forbidden for ***this*** challenge. All neural networks must be trained from scratch, without relying on any pre-existing architectures or pretrained weights.


# 2ND Challenge - Histological Image Classification

- Developing a Convolutional Neural Network (CNN) for Image Classification tasks, performing Transfer-Learning and Fine-Tuning

In this assignment, a collection of images was given. Each image comes from a low-magnification WSI of human tissue. The objective of this task is to perform classification — that is, to assign each image to one of four possible classes corresponding to molecular subtypes identifying a potential disease within the tissue.
